# Installation

## Stable release

To install Infernal Engine, run this command in your terminal:

```sh
uv add infernal-engine
```

Or if you prefer to use `pip`:

```sh
pip install infernal-engine
```

## From source

The source files for Infernal Engine can be downloaded from the [Github repo](https://github.com/CuriousCactus/infernal_engine).

You can either clone the public repository:

```sh
git clone git://github.com/CuriousCactus/infernal_engine
```

Or download the [tarball](https://github.com/CuriousCactus/infernal_engine/tarball/master):

```sh
curl -OJL https://github.com/CuriousCactus/infernal_engine/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd infernal_engine
uv pip install .
```
